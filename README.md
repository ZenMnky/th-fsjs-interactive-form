# th-fsjs-interactive-form
TreeHouse Full Stack JavaScript Techdegree - Unit 3 Project - Interactive Form

## Description 

An interactive registration form for a fictional conference called "FullStack Conference"  
[Live Link](https://zenmnky.github.io/th-fsjs-interactive-form/)


Used Vanilla JavaScript and custom RegEx to:  
- Add customized and conditional behavior and interactivity to a form
- Validate user input and provide helpful error messages when the user enters invalid information into the form fields.

## Technologies
HTML5, CSS3, JS
